API Documentation

Complete documentation for all available endpoints

GET
/api/ssweb?url={website_url}

This endpoint captures a screenshot of the specified website and returns it as an image.

Parameter Type Required Description
url String Yes The URL of the website to capture (must include http:// or https://)
// Example usage
fetch('https://generich.my.id/api/ssweb?url=https://google.com')
.then(response => response.blob())
.then(blob => {
  const img = document.createElement('img');
  img.src = URL.createObjectURL(blob);
  document.body.appendChild(img);
});
