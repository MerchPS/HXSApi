API Endpoints

Complete documentation for all available endpoints

GET
/api/ssweb
Public

Capture screenshots of any website and receive the image directly.

Parameters

Name
Type
Required
Description
url
String
Yes
Website URL to capture (include http:// or https://)

Example Request

fetch('https://your-api.vercel.app/api/ssweb?url=https://google.com')
  .then(response => response.blob())
  .then(blob => {
    const img = document.createElement('img');
    img.src = URL.createObjectURL(blob);
    document.body.appendChild(img);
  });
GET
/api/health
Public

Check API health status and server information.

Example Response

{
  "success": true,
  "message": "API berjalan dengan baik",
  "timestamp": "2023-11-15T10:30:00.000Z",
  "version": "1.0.0"

POST
/api/developer/logs
Protected

Access detailed API logs and monitoring data. Requires authentication.

Parameters

Name
Type
Required
Description
password
String
Yes
Developer password for authentication
limit
Number
No
Number of log entries to return (default: 50)

Example Request

fetch('/api/developer/logs', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    password: 'your_password',
    limit: 100
  })
})
.then(response => response.json())
.then(data => console.log(data));

Usage Examples

Practical examples for integrating HXS API into your projects

JavaScript

// Basic usage
const imgUrl = 
  '/api/ssweb?url='
  + encodeURIComponent(url);

document.getElementById('screenshot')
  .src = imgUrl;

HTML Direct

<img src="/api/ssweb?url=https://example.com" 
     alt="Website Screenshot"
     onerror="this.style.display='none'">

React Component

function Screenshot({ url }) {
  const [src, setSrc] = useState('');
  
  useEffect(() => {
    setSrc(`/api/ssweb?url=${
      encodeURIComponent(url)
    }`);
  }, [url]);

  return <img src={src} />;
}
