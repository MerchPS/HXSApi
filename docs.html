Base URL: https://hxs-apis.vercel.app/api

Introduction

HXS API menyediakan layanan untuk mengambil screenshot dari website secara mudah dan cepat. API ini dirancang untuk developer yang membutuhkan fitur capture website dengan kualitas tinggi.

Catatan: Semua endpoint mendukung CORS dan dapat diakses dari domain mana pun.

SSWeb API - Website Screenshot

Endpoint untuk mengambil screenshot dari website tertentu. Mendukung berbagai jenis website dengan hasil berkualitas tinggi.

GET /api/ssweb?url=https://example.com
url *required

URL lengkap website yang akan di-screenshot (harus termasuk protocol http:// atau https://)

Response Success:

200 OK

Mengembalikan gambar dalam format JPEG

Content-Type: image/jpeg
Content-Length: [size]
X-Response-Time: [time]ms

Response Error:

400 Bad Request 500 Internal Server Error 
{
  "success": false,
  "message": "Pesan error",
  "error": "Detail error"
}

Contoh Penggunaan:

// JavaScript
const response = await fetch('https://hxs-apis.vercel.app/api/ssweb?url=https://google.com');
const imageBlob = await response.blob();

// Gunakan di elemen img
const img = document.createElement('img');
img.src = URL.createObjectURL(imageBlob);
document.body.appendChild(img);

Error Handling

Semua endpoint mengikuti format error handling yang konsisten untuk memudahkan debugging.

Format Error Standar:

{
  "success": false,
  "message": "Deskripsi error yang mudah dipahami"
}

Status Code yang Digunakan:

200 Success - Request berhasil

400 Bad Request - Parameter tidak valid atau tidak lengkap

405 Method Not Allowed - Method HTTP tidak didukung

500 Internal Server Error - Kesalahan server internal

CORS Support

Semua endpoint mendukung Cross-Origin Resource Sharing (CORS) dengan konfigurasi berikut:

Access-Control-Allow-Origin: *
Access-Control-Allow-Methods: GET, OPTIONS
Access-Control-Allow-Headers: Content-Type
Catatan: API dapat diakses dari domain mana pun (*) dan mendukung preflight OPTIONS requests untuk keamanan CORS.
