Introduction

Welcome to HXS API Platform - Powerful APIs for modern developers

HXS API Platform provides a comprehensive suite of APIs designed to help developers build amazing applications faster. From website screenshots to media processing, we handle the complex infrastructure so you can focus on building your product.

Key Features

  • RESTful Design - Clean, consistent API design following REST principles
  • JSON Responses - All responses in JSON format for easy parsing
  • HTTPS Only - Secure connections with SSL encryption
  • CORS Enabled - Cross-origin requests supported for web applications
  • Rate Limiting - Fair usage policies to ensure service stability
  • Real-time Monitoring - Live status updates and performance metrics

Getting Started

Start integrating HXS APIs in just a few minutes

Base URL

All API requests should be made to:

https://your-domain.vercel.app/api

Making Your First Request

Test the API with a simple health check:

fetch('https://your-domain.vercel.app/api/health')
  .then(response => response.json())
  .then(data => console.log(data));

Authentication

Secure your API requests with authentication

HXS API uses API keys to authenticate requests. You can view and manage your API keys in the Developer Console.

Getting Your API Key

Access the Developer Console to generate and manage your API keys:

Using Your API Key

Include your API key in the request header:

fetch('https://your-domain.vercel.app/api/endpoint', {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY'
  }
})
.then(response => response.json())
.then(data => console.log(data));

SSWeb API

Capture high-quality screenshots of any website

Website Screenshot API Live

Capture screenshots of any website with customizable options. Perfect for generating previews, monitoring changes, or creating content.

Capture Screenshot

GET
/api/ssweb

Capture a screenshot of the specified website and receive the image directly.

Query Parameters
Name
Type
Required
Description
url
String
Yes
Website URL to capture (include http:// or https://)
width
Number
No
Viewport width in pixels (default: 1280)
height
Number
No
Viewport height in pixels (default: 720)
Example Request
fetch('https://your-domain.vercel.app/api/ssweb?url=https://google.com')
  .then(response => response.blob())
  .then(blob => {
    const img = document.createElement('img');
    img.src = URL.createObjectURL(blob);
    document.body.appendChild(img);
  });

YouTube API

Extract data and information from YouTube videos

YouTube Data API Live

Get video information, download videos, and extract metadata from YouTube. Perfect for content analysis, archiving, or integration with your applications.

Get Video Information

GET
/api/youtube/info

Retrieve detailed information about a YouTube video including title, description, duration, and more.

Query Parameters
Name
Type
Required
Description
url
String
Yes
YouTube video URL or ID
Example Request
fetch('https://your-domain.vercel.app/api/youtube/info?url=https://youtube.com/watch?v=VIDEO_ID')
  .then(response => response.json())
  .then(data => console.log(data));
