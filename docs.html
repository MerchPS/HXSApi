Introduction

HXS API menyediakan berbagai layanan untuk pengembangan aplikasi, termasuk capture screenshot website, status monitoring, dan health check.

Base URL: https://your-domain.com/api

SSWeb API - Website Screenshot

Endpoint untuk mengambil screenshot dari website tertentu.

GET /api/ssweb?url=https://example.com
url *required

URL lengkap website yang akan di-screenshot (harus termasuk protocol http:// atau https://)

Response Success:

200 OK

Mengembalikan gambar dalam format JPEG

{
Content-Type: image/jpeg
Content-Length: [size]
X-Response-Time: [time]ms
}

Response Error:

400 Bad Request 500 Internal Server Error 
{
  "success": false,
  "message": "Pesan error",
  "error": "Detail error"
}

Contoh Penggunaan:

// JavaScript
const response = await fetch('/api/ssweb?url=https://google.com');
const imageBlob = await response.blob();

Error Handling

Semua endpoint mengikuti format error handling yang konsisten.

Format Error Standar:

{
  "success": false,
  "message": "Deskripsi error yang mudah dipahami"
}

Status Code yang Digunakan:

200 Success - Request berhasil

400 Bad Request - Parameter tidak valid atau tidak lengkap

405 Method Not Allowed - Method HTTP tidak didukung

500 Internal Server Error - Kesalahan server internal

CORS Support

Semua endpoint mendukung Cross-Origin Resource Sharing (CORS) dengan konfigurasi berikut:

Access-Control-Allow-Origin: *
Access-Control-Allow-Methods: GET, OPTIONS
Access-Control-Allow-Headers: Content-Type
Catatan: API dapat diakses dari domain mana pun (*) dan mendukung preflight OPTIONS requests.

Logging System

Setiap request ke API akan tercatat secara otomatis dengan informasi berikut:

  • IP Address pengguna
  • HTTP Method yang digunakan
  • Endpoint yang diakses
  • Status code response
  • User-Agent browser/klien
  • Waktu respons (response time)
  • Pesan error (jika terjadi)
  • URL target (untuk endpoint ssweb)
